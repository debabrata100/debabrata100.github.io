Demonstration of using cooperatively scheduled background tasks using the requestIdleCallback() method.

Enable requestIdleCallback to make user input responsive
Playground
  • Long Task
  • In the Playground section, click on + button, you will be able to add circles. Now click on start long task in the Long Task section, you will notice playground section is not interactive. Wait for 20s, you will notice progressbar will show 100% at once.
  • Now check the checkbox(Enable requestIdleCallback to make user input responsive) and repeat the above step, you will notice everything is interactive. Even the progressbar start moving.
